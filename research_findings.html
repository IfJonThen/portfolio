    Summary of Findings

    • Based on my tests, what I was able to draw from the results was that my tool and web page is very bland and rather visually unappealing. Based on some suggestions, I was told that changing the design of the page would help greatly. In addition. prior to seeing the results of the page, I thought that my tools page was easy to use, however based on some feed back that I recieved, it seems that some of my buttons on the tools page could have some additional instructions included to help clarify their purpose to the user

    • from my cognitive walkthroughs I was able to find that my web page was easy to navigate however, I was told that parts of my website was not consistent with a uniform look. In addition, due to the limitation of the click test from UsabilityHub.com, the testers were only able to click and not actually interact with the web page. Once I present the web page to people to use, they were able to use it, contrary to my results received from my usabilityhub test.

    • Overall, I knew that my tools and portfolio site is far from a complete perfected project however, the tests was able to help me understand that I still have alot of work to do especially in the aesthetics department. I may need to look into redesigning the tools section of my website and doing additional research into learning about visual design. Visual design seems to be a common topic when reading about the criticisms in my usability test.

    Implementation of Findings

    • Moving forward, I plan on doing additional research on how to make my website and the tools section to be better designed so that people will find it more visually appealing. I am not an artsy person so I think that this research will be very useful in helping me design things in the future.

    • Because some people thought that my tools contained some unfamiliar buttons, I thought about adding hints over my buttons however, I need to think of a better way to implement that since hover doesnt work on mobile. However I plan on adding more instructions to help clarify the usage of my tool since confusion was a common response in my usabilityhub tests

    • Some difficulty and challenges that I faced was that alot of times, I knew what changes in the design and what additional content that I wanted to make, however when it came down to implementing it, it often became much more difficult especially when I was sharing a similar css file with 4-5 different pages. To counteract this, I had to write specific CSS for the pages that I needed.
